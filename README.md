# Personal Website

## Description
This project is a personal portfolio website designed to showcase information about me, my background, and my personal projects. The website is built using HTML, CSS, and JavaScript and includes multiple pages for navigation and organization.

## Pages Included

### Home Page
The home page serves as the main landing page for visitors. It contains a welcome message, navigation menu, and an overview of the website.

### About Page
The about page provides background information about me, including my education, interests, skills, and career goals.

### Personal Projects Page
The personal projects page highlights projects I have worked on. Each project includes a title, description, technologies used, and screenshots or links when available.

## Features
- Multi-page website design
- Responsive navigation menu
- Clean and modern layout
- Project showcase section
- Easy-to-read formatting

## Technologies Used
- HTML5
- CSS3

## Installation
1. Download or clone the project files.
2. Open the project folder.
3. Open `index.html` in a web browser.

## Usage
Use the navigation menu to move between the Home, About, and Personal Projects pages.
